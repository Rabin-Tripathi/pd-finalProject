# Fun Olympics 2023 Live Broadcasting Website

Welcome to the Fun Olympics 2023 Live Broadcasting Website repository. This project is a web application built using HTML, CSS, JavaScript, Bootstrap, jQuery, PHP, and MySQL. 
The website allows registered users to access live broadcasts, receive notifications, engage in live chat, and provides admin functionalities for managing users, broadcasts, news articles, and categories.

## Installation

1. Clone the repository to your local machine using `git clone https://github.com/Rabin-Tripathi/pd-finalProject.git`.
2. Ensure you have a local development environment set up with PHP and MySQL installed.
3. Import the included SQL database file `fun_olympics.sql` to set up the necessary database structure and sample data.
4. Update the database connection credentials in the `config.php` file with your MySQL username, password, and database name.
5. Place the project files in your web server's document root directory.
6. Access the website through your web browser by visiting `http://localhost/pd-finalProject`.

## Usage

1. Register a new account on the website to gain access to the full range of features.
2. Login to your account to start exploring the live broadcasts and utilizing the personalized functionalities.
3. Customize your notification settings to receive updates on upcoming broadcasts.
4. Engage with other users through the live chat feature.
5. Admin users can access the admin panel by visiting `http://localhost/pd-finalProject/admin` and logging in with the provided admin credentials.
6. In the admin panel, manage user accounts, create/edit/delete broadcasts, news articles, and categories.

## Folder Structure

- `css/` - Contains CSS files for styling the website.
- `js/` - Contains JavaScript files for client-side functionalities.
- `includes/` - Contains reusable PHP files for database connection and common functions.
- `admin/` - Contains the admin panel files.
- `images/` - Contains images used in the website.
- `config.php` - Configuration file for database connection.
- `index.php` - Homepage of the website.
- `login.php` - User login page.
- `register.php` - User registration page.
- `dashboard.php` - User account dashboard page.
- `broadcasts.php` - Page for viewing and selecting broadcasts.
- `chat.php` - Live chat page.
- `admin.php` - Admin login page.
- `admin_dashboard.php` - Admin dashboard page.
- `admin_users.php` - User management page.
- `admin_broadcasts.php` - Broadcast management page.
- `admin_news.php` - News management page.
- `admin_categories.php` - Category management page.
